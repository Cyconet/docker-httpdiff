Builds a docker image with the [httpdiff](https://github.com/jgrahamc/httpdiff) ready to run.

Running
-------

- run the docker container with:

```
docker run --rm -it waja/httpdiff --help 
```

Building
--------

```
make build
```

Get a shell in a running container
----------------------------------

```
make shell
```
